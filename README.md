# Legal Hub MCP - Argentina

Hub MCP unificado para abogados argentinos. Conecta Claude Desktop con 11 fuentes jurídicas oficiales a través de un único servidor.

## Conectores incluidos (168 tools)

| Conector | Fuente | Tools |
|----------|--------|-------|
| bopba | Boletín Oficial PBA | 15 |
| bora | Boletín Oficial Nacional | 14 |
| infoleg | InfoLEG - Leyes Argentinas | 20 |
| juba | JUBA - SCBA | 21 |
| normativapba | Normativa PBA | 9 |
| pjn | Poder Judicial de la Nación | 16 |
| pjnjuris | PJN Jurisprudencia CAF | 20 |
| ptn | Procuración del Tesoro | 22 |
| saij | SAIJ | 11 |
| scba | SCBA | 4 |
| tfn | Tribunal Fiscal de la Nación | 16 |

## Requisitos

- Node.js v18 o superior
- Claude Desktop
